# Brandquist IT Website

A professional, responsive website for IT services featuring Home, Services, and Contact sections.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Contact Form**: Functional contact form with EmailJS integration
- **Navigation**: Smooth scrolling navigation with mobile hamburger menu
- **SEO Ready**: Semantic HTML structure and proper meta tags
- **Fast Loading**: Optimized CSS and JavaScript for quick page loads

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive functionality and form handling
- **EmailJS**: Contact form email delivery service

- **Google Fonts**: Inter font family for typography

## Setup Instructions

### Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. No build process required - it's a static website

### Deployment Options

#### GitHub Pages
1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Your site will be available at `https://yourusername.github.io/repository-name`

#### Other Hosting Services
- Upload `index.html`, `styles.css`, `script.js`, and logo files to your web hosting service
- Ensure `index.html` is in the root directory

## Customization

### Contact Form Setup
The contact form uses EmailJS service. To set up:

1. Create an account at [EmailJS](https://emailjs.com)
2. Set up an email service and template
3. Update the EmailJS configuration in `script.js`:
   - Replace `YOUR_PUBLIC_KEY` with your EmailJS public key
   - Replace `YOUR_SERVICE_ID` with your service ID
   - Replace `YOUR_TEMPLATE_ID` with your template ID

### Styling
- Modify `styles.css` to change colors, fonts, or layout
- Current color scheme uses blue (#3498db) as primary color
- Font family is set to 'Inter' from Google Fonts

### Content
- Update business information in `index.html`
- Replace service descriptions with your offerings
- Add your own logo and branding
